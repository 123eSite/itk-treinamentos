## 2. Cores

### Azul (primária e variações)

Base primária: **#162947**. Variações do azul, do mais claro ao mais escuro:

| Nome | Hex | Uso sugerido |
|------|-----|--------------|
| Azul primário | `#162947` | Marca, destaque |
| Azul 2 | `#022d66` | Fundos, gradiente |
| Azul 3 | `#00214c` | Gradiente, elementos escuros |
| Azul 4 | `#00204B` | Mais escuro, contraste |

### Neutros

| Nome | Hex |
|------|-----|
| Preto | `#000000` |
| Branco | `#ffffff` |

### Cinza (fundos e textos)

Cerca de 5 variações para fundos e textos:

| Nome | Hex | Uso sugerido |
|------|-----|--------------|
| Cinza 1 | `#f5f5f5` | Fundo claro |
| Cinza 2 | `#e8e8e8` | Bordas, divisórias |
| Cinza 3 | `#9e9e9e` | Texto secundário |
| Cinza 4 | `#616161` | Texto terciário |
| Cinza 5 | `#212121` | Texto em fundos claros |

### Gradiente

- **De:** `#022D66`
- **Para:** `#00214C`

Usar em CTAs, headers ou blocos de destaque.

## 3. Tipografia

### Fontes

| Uso | Fonte | Observação |
|-----|--------|------------|
| **Texto (corpo)** | Open Sans | Disponível no [Google Fonts](https://fonts.google.com/specimen/Open+Sans) |
| **Títulos** | Montserrat | Disponível no [Google Fonts](https://fonts.google.com/specimen/Montserrat) |

### Tamanhos (escala proporcional)

Base: **parágrafo 16px**. Demais tags proporcionais; legenda/caption menor.

| Elemento | Tamanho | Fonte |
|----------|---------|--------|
| `p` | 16px | Open Sans |
| Legenda / texto menor | 14px | Open Sans |
| `h6` | 18px | Montserrat |
| `h5` | 20px | Montserrat |
| `h4` | 24px | Montserrat |
| `h3` | 28px | Montserrat |
| `h2` | 32px | Montserrat |
| `h1` | 36px | Montserrat |

## 4. Espaçamentos

### Malha base

Sistema espacial baseado no **8**: grade de 8px. Valores proporcionais:

- **8** — 8px
- **16** — 16px
- **24** — 24px
- **32** — 32px
- **64** — 64px
- **128** — 128px

Usar esses valores para margens, paddings e gaps (ex.: 8, 16, 24, 32, 64, 128).

### Entre seções

- **Espaçamento vertical entre seções (top e bottom):** sempre **100px**.
